Daten herunterladen

Falls Sie eine eigene plantwatch-Instanz betreiben möchten, ohne plantwatch-data zu benutzen, finden Sie die nötigen Dateien hier zum Download:

  • plantwatch.db
  • data.zip

Wünschen Sie eine postgres-basierte Installation, benötigen Sie data.zip, wobei Sie die enthaltenen Dateien in den db-Ordner kopieren müssen.
Wünschen Sie eine sqlite-basierte Installation, benötigen Sie plantwatch.db, wobei Sie diese in den Ordner plantwatch (den mit manage.py) kopieren müssen.
